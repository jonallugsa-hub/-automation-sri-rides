# SRI Invoice (RIDE) Download Automation
# Automatización de descarga de comprobantes SRI (RIDE)

---

## 🇺🇸 Description
Python script that automates the search and download of electronic invoices (RIDE) from the Ecuadorian SRI portal using Excel data.

This solution reduces manual work and improves efficiency in administrative processes.

---

## 🇪🇸 Descripción
Script en Python que automatiza la búsqueda y descarga de comprobantes electrónicos (RIDE) desde el portal del SRI utilizando datos desde Excel.

Reduce trabajo manual y mejora la eficiencia en procesos administrativos.

---

## ⚙️ Technologies
- Python
- Selenium
- Pandas

---

## 🚀 Features

- Automated navigation in SRI portal
- Invoice search using Excel data
- Automatic PDF download
- File renaming and organization
- Result tracking (found / not found)
